### Float to Int

```c
int b = 3.1;
printf("%d \n", b); //3
printf("%f \n", b); //0.0
```

```c
float c = 3.1;
printf("%d \n", c); //-1073741824
```

### Type cast

```c
float a = (float) 3;
printf("%d \n", a);	//0
printf("%f \n", a); //3.000
```    

### Calculation

```c
int a = (998/1024)*100; // a = 0
```
